# Task Management API

## Tech Stack
- FastAPI
- Python
- Uvicorn

## Features
- Create Task (POST)
- Read Tasks (GET)
- Update Task (PUT)
- Delete Task (DELETE)

## How to Run
1. Activate venv
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   uvicorn app.main:app --reload
4. Open:
   http://127.0.0.1:8000/docs
